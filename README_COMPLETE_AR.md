# VoltEye Complete Precision System

هذه النسخة موجهة لحل مشكلة خلط الأجهزة في التحNight.

## ما الذي تغيّر؟
- لكل جهاز الآن **بصمة faultImpact مستقلة**
- حقن العطل في Simulation أصبح يطبّق **أثر الجهاز نفسه** بدل زيادة عامة ثابتة
- التحNight أصبح يعتمد على:
  1. **نوع العطل المتوقع** من residual
  2. **مطابقة بصمة القراءة** مع بصمة كل جهاز نشط
  3. دعم خفيف فقط من التعلم الذاتي، وليس تحكمًا كاملًا
- Fault Location يُستنتج من:
  - أفضل تطابق
  - ثم تصويت أعلى المرشحين

## لماذا هذا أدق؟
لأن النظام لم يعد يفضّل الجهاز الكبير فقط، بل يقارن:
- فرق Current
- فرق Leakage
- فرق THD
- فرق PF
- فرق Voltage

مع ما يفترض أن يسببه كل جهاز إذا تعطّل.

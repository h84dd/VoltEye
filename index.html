About Project

VoltEye AI+ is an electrical fault diagnosis system that runs locally in the browser.

  • Stable, demo-ready simulation mode
  • Real reading mode via API or WebSocket
  • Fault-signature-based analysis
  • Self-learning from user confirmations and corrections
  • Local case storage in LocalStorage
